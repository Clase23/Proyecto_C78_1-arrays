
1. Objetivo del Proyecto
Desarrollar una aplicación en JavaScript que utilice arrays para almacenar, manipular y visualizar datos. El propósito es comprender el uso de arrays y sus métodos principales para trabajar con conjuntos de información de manera eficiente.


2. Definición y Conceptos Básicos
Array: Un array en JavaScript es una estructura de datos que permite almacenar múltiples valores en una sola variable.

Índices: Cada elemento en un array tiene un índice numérico que comienza desde 0.

Métodos de Arrays: JavaScript proporciona varios métodos para manipular arrays, como push(), pop(), shift(), unshift(), map(), filter(), reduce(), entre otros.

Iteración: Se pueden recorrer los arrays con bucles como for, forEach(), o map().

3. Alcance
El proyecto incluirá la creación de un programa que realice operaciones básicas con arrays, tales como:

Creación y modificación de arrays.

Agregar y eliminar elementos.

Ordenar y filtrar datos.

Recorrer arrays y realizar operaciones con sus elementos.

Usa editor de código (Visual Studio Code).

Un navegador web para ejecutar y probar el código.

6. Criterios de Evaluación
Correcta implementación y manipulación de arrays.
